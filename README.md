# Sales-Data-Analytics-Using-Python-Pandas-Tableau

A beginner-friendly data analytics project that uses Python Pandas to analyze real-world sales data stored in CSV format. The project demonstrates how raw business data can be cleaned, transformed, analyzed, and visualized efficiently to generate meaningful insights.
This project focuses on sales data analysis, including revenue calculation, performance trends, discount analysis, product insights, and dashboard storytelling using Tableau.

# Technologies Used

1. Python
2. Pandas
3. NumPy
4. Matplotlib
5. Tableau
6. CSV Dataset

# Flow of Project

Raw sales CSV data
-> Data Cleaning & Preparation
-> Feature Engineering
-> Business Analytics
-> Data Visualization
-> Tableau Dashboard
-> Insight Reporting
